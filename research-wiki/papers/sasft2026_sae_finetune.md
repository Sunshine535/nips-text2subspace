# One-line thesis

Uses SAE feature analysis to guide fine-tuning decisions; validates that SAE features capture meaningful adapter effects.

## Key Results

- SAE features reliably capture fine-tuning effects
- Feature-guided fine-tuning improves over naive fine-tuning

## Relevance to This Project
